# 💌 For You, My Love

তোমার GF-এর জন্য বানানো একটা কাস্টম romantic website।

## 📁 Folder Structure
```
love-website/
├── index.html
├── style.css
├── script.js
└── assets/
    ├── photo1.jpg  →  photo6.jpg    (6 টা ছবি)
    ├── our-video.mp4                 (একটা ভিডিও)
    ├── video-poster.jpg              (ভিডিওর thumbnail — optional)
    └── music.mp3                     (background music — optional)
```

## 🚀 কীভাবে চালাবে
১. পুরো `love-website` folder টা download করে নাও।
২. `assets/` folder এ তোমার ছবি, ভিডিও, গান রাখো — ঠিক উপরের নাম গুলোর মতো।
৩. `index.html` file টা browser এ open করলেই হবে। ব্যস! ✨

## ✏️ কী কী customize করতে পারো
- **চিঠির লেখা**: `index.html` এর `<section class="letter-section">` এর ভেতরের text।
- **ছবির নিচের ক্যাপশন**: `<div class="polaroid">` এর `<p>` tag গুলো।
- **Reasons I love you**: `<div class="flip-back"><p>...</p></div>` — ৮ টা কার্ডের text।
- **নাম**: "My Dearest" → ওর নাম দিয়ে replace করতে পারো।

## 🌐 Online হোস্ট করতে চাইলে
